Customer Segmentation Using Shopping Trends Data

Project Overview

    1. This project focuses on analyzing shopping trends data to understand customer purchasing behavior and identify different customer segments.      
    2. Basic clustering techniques were applied to group customers based on shopping patterns, spending behavior, and discount usage. 
    3. The project also includes exploratory data analysis and business recommendations for targeted marketing.

Objective

    1. Analyze customer shopping behavior
    2. Study discount impact on product categories
    3. Analyze card spending patterns across age groups and locations
    4. Perform basic customer segmentation using clustering
    5. Generate business insights for targeted marketing strategies

Dataset Information

    Dataset: Shopping Trends Dataset
    Total Records: 3000
    Total Attributes: 19
    Type: Structured customer transaction data
   
Tools & Technologies Used

    1. Python (Pandas)
    2. NumPy
    3. Scikit-learn (Basic K-Means Clustering)
    4. Jupyter Notebook
    5. MySQL and Excel
   
Data Processing & Cleaning

    1. Loaded and processed data using Pandas
    2. Checked for missing values and inconsistencies
    3. Performed categorical data encoding
    4. Prepared data for analysis and clustering

Exploratory Data Analysis
Discount Analysis

    1. Analyzed product categories with higher discount impact
    2. Observed higher purchase trends in footwear products during discounts
    
Card Spending Analysis

    1. Compared spending behavior across different age groups
    2. Customers aged above 45 showed higher card spending patterns
    
Seasonal & Location Analysis

    1. Higher spending observed during Winter and Spring seasons
    2. Urban customers showed increased card usage compared to other locations
   
Clustering & Segmentation

    Basic customer segmentation was performed using K-Means clustering.
    Customer Segments Identified
       1. Cluster 0 – High-Value Loyal Customers
       2. Cluster 1 – Discount-Sensitive Customers
       3. Cluster 2 – Low-Frequency Customers
   
Business Recommendations

    1. Provide loyalty rewards for high-value customers
    2. Offer discounts and coupons for price-sensitive customers
    3. Create re-engagement campaigns for low-frequency customers
    4. Focus seasonal promotions during high-spending periods
